Terminal Azul (PowerShell) -> swipl archivo.pl. -> Abrir Prolog y cargar el código.
Dentro de Prolog (?-) -> consult('archivo.pl'). -> Cargar el código si ya entraste a Prolog.
Dentro de Prolog (?-) -> resolver(matriz(...)). -> Ejecutar tu programa.
Dentro de Prolog (?-) ->  halt. ->  Salir de Prolog y volver a la terminal.
